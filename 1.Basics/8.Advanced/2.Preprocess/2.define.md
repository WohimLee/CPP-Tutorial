# define
## 1 宏定义常量
>语法
- 它的作用是定义一个叫作 name 的宏定义，然后每当在程序中遇到这个名字的时候，它就会被 value 代替
```c++
#define name value
```

>Example
```c++
#include <iostream>
using namespace std;

#define NUM 5

int main()
{
    char array[NUM];
    for (int i; i < NUM; i++)
    {
        array[i] = i;
    }
    
    return 0;
}
```

## 2 宏定义函数
>Example
```c++
#include <iostream>
using namespace std;

// #define getmax(a, b) a > b ? a : b
#define getmax(a, b)  ((a) > (b)) ? (a) : (b)

int main()
{
    int x = 5, y;
    y = getmax(x, 2);
    cout << y << endl;
    return 0;
}
```

# undef
#undef 完成与 #define 相反的工作，它取消对传入的参数的宏定义
>Example
```c++
#include <iostream>
using namespace std;

#define MAX_WIDTH 100
#undef MAX_WIDTH
#define MAX_WIDTH 200

int main()
{
    cout << MAX_WIDTH << endl;
    return 0;
}
```
