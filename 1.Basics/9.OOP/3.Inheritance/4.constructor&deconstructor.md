# 构造和析构顺序
子类继承父类后，当创建子类对象，也会调用父类的构造函数
- 思考: 父类和子类的构造和析构顺序是谁先谁后？

>示例
```c++
class Base
{
public :
    Base(){cout << "Base类构造函数" << endl;} 
    ~Base(){cout << "Base类析构函数" << endl;}
};

class Son : public Base{
public:
    Son(){cout << "Son类构造函数" << endl;}
    ~Son(){cout << "Son类析构函数" << endl;}
};

int main()
{
    Son a;
    return 0;
}
```
